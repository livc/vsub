>>>>>>>>>>>>
Installation
>>>>>>>>>>>>
Installtion is easy. Just run the following command
::

    pip install git+https://github.com/livc/vsub


>>>>>>>
Example
>>>>>>>
* typical

::

    vsub "python xxx" --node 2

*   If you want to run a shell(It is ok if no name)

::

    vsub example.sh --name test --node 2 --shell
